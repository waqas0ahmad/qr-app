# Back-end server

## Pre-requisites

- Install npm and nodeJs `https://nodejs.org/`

$ node --version
- Must be `npm version >= 6.x`

$ npm --version
- Must be `npm version >= 12.x`

- Install required dependencies
$ npm install

### Run Node Server

$ npm start
